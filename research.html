Status Categories (mirror Notion)
⚪ Researching

Names you're investigating but haven't committed to. Sunday sessions land here first, then move up or get rejected.

🟡 Pending Entry

Setup validated, waiting for entry trigger. This is where Sunday's 2–3 finalists live. Alert set at entry level.

🟢 Live Position

Entered. Bracket order active. Managed per the Live Trading rules (4H/Daily monitoring, no 1-min).

🔵 Exited

Closed. Log the outcome in Journal tab. Lessons captured.

🔴 Rejected

Setup invalidated before entry, OR chart-read failed the LJ method. Don't feel bad about rejects — protects capital.

📝 Paper Position

Test trades while learning a new setup (reversal plays, new sector, new broker workflow). No real capital.

Workflow — From Idea to Play
  1. Sunday session or your own screener surfaces a name → add to Notion as Researching
  2. Chart-read on TradingView (4H + Daily for swings · use EMAs 9/20/50 for options)
  3. If LJ method confirms (3 of 4 indicators aligned + S/R + no earnings in DTE) → move to Pending Entry
  4. Set Webull/broker price alert at entry level
  5. Alert triggers → confirm 1-min setup → place bracket order → move to Live Position
  6. Trade closes (target · stop · manual invalidation) → move to Exited · log in Journal
